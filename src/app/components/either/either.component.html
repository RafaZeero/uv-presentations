What is a Data type?

A data type is an abstraction that encapsulates one reusable coding pattern.

What is Either?

It is the result of a computation that can fail. Can solve the limitation that the Option type has.

  • None ----------> Left<E>
  • Some<A> ---> Right<A>

Either is an entity whose value can be of two different types, called left and right.

  • Left: Error path
  • Right: Success path

How is the Either typed?

type Either<E, A> = Left<E> | Right<A> type Left<E> = { _tag: 'Left'; left: E; }; type Right<A> = { _tag: 'Right'; right: A; };

Basic Constructors

const left = <L, R = never>(e: L): Either<L, R> => ({ _tag: 'Left', left: e }); const right = <R, L = never>(a: R): Either<L, R> => ({ _tag: 'Right', right: a });

Guards

const isLeft = <L, R>(ma: Either<L, R>): ma is Left<L> => ma._tag === 'Left'; const isRight = <L, R>(ma: Either<L, R>): ma is Right<R> => ma._tag === 'Right';
